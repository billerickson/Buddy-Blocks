# Unit 2: Reference Tools And Meaning Fit

Track: Vocabulary
Level: Grade 3
Stage: Unit Design Brief
Source: `research/grade-03-vocabulary/03-course-map.md`

## 1. Unit Goal

Students use glossaries, beginning dictionaries, and simplified digital-style entries to verify word meaning. By the end of the unit, students should choose the definition that fits a sentence and explain how a reference entry confirms or corrects their first meaning guess.

## 2. Why This Unit Belongs Here

Reference skills belong early because context clues are useful but sometimes incomplete. Students need to learn that a dictionary or glossary is not a separate activity; it is part of a word-solving cycle that begins with the sentence and ends with checking meaning fit.

## 3. Key Concepts, Vocabulary, Representations, Texts, Forms, Or Models

- Concepts: reference entry structure, multiple definitions, fit to context, verification, pronunciation and syllable support.
- Vocabulary: dictionary, glossary, entry, definition, example sentence, syllable, pronunciation, guide word, multiple definitions, reference tool, verify.
- Representations and models: simplified glossary entries, dictionary-entry cards, context-plus-entry pairs, definition-fit charts.
- Texts and forms: context sentences, entry excerpts, short glossary-based informational paragraphs, one-sentence verification explanations.

## 4. Prerequisite Knowledge

Students need alphabet-order readiness, basic print-reference habits, Unit 1 context-clue routines, and the ability to compare a short definition with a sentence.

## 5. Common Misconceptions

- The first definition is always correct.
- A reference entry can be used without reading the sentence.
- Syllables and pronunciation tell the meaning.
- A glossary is only for rare or science words.
- Checking a reference means the context clue was useless.

## 6. Lesson Sequence Rationale

Start with glossary entries because they are short and tied to one text. Then move to multiple definitions and context fit, which is the most important Grade 3 reference skill. A short pronunciation and syllable lesson supports decoding without overloading students with advanced dictionary apparatus. End with strategy choice: when should a student use context alone, and when should they check a reference?

## 7. Review Connections To Earlier Units

Every lesson should begin with a context guess from Unit 1 before consulting an entry. Students should continue naming the clue that led to their first meaning guess, then compare that guess with the reference definition.

## 8. Preparation For Later Units

This unit prepares Unit 5 multiple-meaning and homograph work, where definition choice matters. It also supports Unit 7 content-area vocabulary, where glossaries, captions, diagrams, and entries become part of academic reading.

## 9. Assessment Targets

- Read a simple glossary or dictionary entry.
- Identify useful entry parts: word, definition, example, syllable, pronunciation support.
- Select the definition that fits a context sentence.
- Explain why a non-fitting definition should be rejected.
- Decide whether context, a reference check, or both are needed.

## 10. Recommended Lesson Titles

1. **Read A Glossary Entry**: Use a short entry to connect a word, definition, and example sentence.
2. **Choose The Definition That Fits**: Select the correct meaning when one word has more than one definition.
3. **Use Syllables And Pronunciation Supports**: Use entry supports to read a word without confusing pronunciation support with meaning.
4. **Decide When To Check A Reference**: Choose when context is enough and when a glossary or dictionary should verify meaning.
