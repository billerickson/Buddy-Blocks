# Unit 8: Equivalent Fractions And Fraction Comparison

Track: Math
Level: Grade 3
Stage: Unit Design Brief
Source: `research/grade-03-math/03-course-map.md`

## 1. Unit Goal

Students reason about equivalent fractions and compare Grade 3 fractions using visual models, number lines, benchmarks, same numerator reasoning, same denominator reasoning, unit size, and same-whole constraints.

## 2. Why This Unit Belongs Here

This unit follows the first fraction unit because equivalence and comparison require students to already understand fractions as quantities. It deepens fraction magnitude before measurement and geometry contexts ask students to use fractions flexibly.

## 3. Key Concepts, Vocabulary, Representations, Texts, Forms, Or Models

- Concepts: equivalent fractions as same quantity, comparison with same denominator, comparison with same numerator, unit fraction size, benchmarks, same whole.
- Vocabulary: equivalent fraction, same quantity, compare, greater than, less than, equal to, same numerator, same denominator, unit size, benchmark, number line, same whole.
- Representations and models: fraction strips, partitioned rectangles, number lines with aligned points, benchmark marks at 0, 1/2, and 1, comparison statements.
- Texts and forms: "Which is greater?" explanations, equivalent-fraction matches, model-based comparison claims, short misconception analysis.

## 4. Prerequisite Knowledge

Students need Unit 7 fraction naming, unit fractions, non-unit fractions, number-line placement, whole-number fractions, numerator/denominator meaning, and same-whole language.

## 5. Common Misconceptions

- Larger denominators mean larger fractions.
- Equivalent fractions look different, so they must be different amounts.
- Diagrams can be compared even when the wholes are different sizes.
- Same numerator and same denominator comparisons use the same reasoning.
- Benchmark fractions are memorized positions, not magnitude references.
- Cross-multiplication or later rules can replace understanding.

## 6. Lesson Sequence Rationale

Begin with equivalence through aligned models and number-line points so students see "same amount" before comparing unequal amounts. Then compare fractions with the same denominator, where same-size parts make numerator reasoning natural. Next compare fractions with the same numerator, where denominator size becomes central. Finish with benchmarks and mixed comparisons that require students to state the reasoning they used.

## 7. Review Connections To Earlier Units

Review Unit 7 equal parts, number-line placement, numerator, denominator, and same whole. Continue light retrieval of multiplication/division facts separately so fact practice remains spaced, but do not use fact rules as the basis for fraction comparison.

## 8. Preparation For Later Units

This unit prepares Unit 9 line plots and measurement marks, Unit 11 equal-area partitions, Unit 12 cumulative fraction transfer, and Grade 4 fraction work with equivalence, ordering, and operations.

## 9. Assessment Targets

- Identify equivalent fractions from models and number lines.
- Explain that equivalent fractions name the same quantity.
- Compare same-denominator fractions by number of equal parts.
- Compare same-numerator fractions by unit fraction size.
- Use same-whole and benchmark reasoning in explanations.

## 10. Recommended Lesson Titles

1. **Equivalent Means Same Amount**: Use models to identify different fraction names for the same quantity.
2. **Equivalent Fractions On Number Lines**: Show equivalent fractions at the same point.
3. **Compare Same Denominators**: Reason from how many same-size parts are counted.
4. **Compare Same Numerators**: Reason from the size of each unit fraction.
5. **Use Benchmarks To Compare**: Compare fractions to 0, 1/2, and 1 when useful.
6. **Same Whole Or Not?**: Decide whether a comparison is valid.
7. **Choose The Comparison Reason**: Match comparison cases to the reasoning that works.
