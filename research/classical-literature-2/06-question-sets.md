## Unit 1: Story Tools For Longer Tales

### Lesson 1: Build A Strong Retelling

```question
key: u01_l01_q01_key_event
type: multiple-choice
prompt: "A tale begins with Mara finding a locked bronze gate. Which detail most belongs in a strong retelling?"
choices:
  - "Mara found a locked bronze gate."
  - "The gate had three scratches near the handle."
  - "Mara's cloak was dark green."
  - "A cricket chirped beside the road."
correctAnswer: "Mara found a locked bronze gate."
explanation: "A locked gate can shape the problem of the story, so it belongs in a retelling."
hint: "Choose the detail that helps explain what changes in the story."
questionGoal: "Select a key event instead of a minor descriptive detail."
misconception: "Including any interesting detail in a retelling."
```

