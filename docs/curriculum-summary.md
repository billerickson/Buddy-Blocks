# Curriculum Summary

This is a content inventory for canonical curriculum tracks, not a child progress report.

Status key:

- `complete`: an authored track folder exists with `track.yaml`, units, and lessons.
- `planned`: the track is described in `docs/v2-curriculum-plan.md` but has not been authored yet.
- `empty`: the expected grade or level slot has no authored or planned track.

Scholastic tracks use academic grade placement. Foundation tracks start every student at Level 1 and unlock later levels in sequence. Foundation levels are stored in grade folders for ordering: Level 1 uses `grade-03`, Level 2 uses `grade-04`, Level 3 would use `grade-05`, and Level 4 would use `grade-06`.
