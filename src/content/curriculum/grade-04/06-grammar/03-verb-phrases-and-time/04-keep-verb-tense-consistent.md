## Teaching Goal

Keep simple verb tense consistent in short contexts unless the meaning clearly changes.

## Student Outcome

The student can choose present, past, or future verb forms that match the time clue in a sentence or two-sentence passage.

## Key Ideas

Verb tense shows time. In a short passage about one time, the verbs should usually stay in that same time.

## Misconception Checks

A verb form is correct by itself even when it shifts the passage time; all past tense forms use `ed`; future tense can be shown without a helping verb.

## Questions

```question
key: u03_l04_q01_past_time_clue
type: fill-blank
prompt: Choose the verb that matches the time clue.
sentenceBefore: "Yesterday, Mia "
sentenceAfter: " to the store."
choices:
  - walks
  - walked
  - will walk
  - walking
correctAnswer: walked
explanation: Yesterday is a past-time clue, so walked fits.
hint: Use the verb form for something that already happened.
```

```question
key: u03_l04_q02_consistent_passage
type: multiple-choice
prompt: Which pair of sentences keeps the same past tense?
choices:
  - We baked bread. We shared it.
  - We baked bread. We share it.
  - We bake bread. We shared it.
  - We will bake bread. We shared it.
correctAnswer: We baked bread. We shared it.
explanation: Both baked and shared tell about the past.
hint: Look for two verbs that happen in the same time.
```

```question
key: u03_l04_q03_fix_tense_shift
type: error-correction
prompt: Correct the tense shift so both actions happened yesterday.
sentence: Yesterday, we walk to the store and bought milk.
acceptedAnswers:
  - Yesterday, we walked to the store and bought milk.
explanation: Bought is past tense, and yesterday tells us walk should be walked.
hint: Match the first verb to the past-time clue.
```

```question
key: u03_l04_q04_present_consistency
type: multi-blank-cloze
prompt: Choose present-tense verbs that keep the sentence consistent.
parts:
  - "Today, Noah "
  - " the map and "
  - " the trail."
blanks:
  - correctAnswer: reads
    choices:
      - reads
      - read
      - will read
  - correctAnswer: follows
    choices:
      - followed
      - will follow
      - follows
explanation: Today can describe what Noah does now, so reads and follows keep the same tense.
hint: Choose two verbs that work in the same time.
```

```question
key: u03_l04_q05_future_helper
type: multiple-choice
prompt: Which sentence uses future tense?
choices:
  - Tara baked muffins.
  - Tara bakes muffins.
  - Tara will bake muffins.
  - Tara is baking muffins now.
correctAnswer: Tara will bake muffins.
explanation: Will bake tells about something that happens later.
hint: Look for the helping verb that points to later.
```

```question
key: u03_l04_q06_identify_shift
type: passage-question
prompt: Read the passage and answer the question.
passageTitle: Tense Check
passage: Last night, Ava packed her bag. She walks to the bus stop early.
question: Which verb should change to keep the past tense consistent?
choices:
  - packed
  - walks
  - bus
  - early
correctAnswer: walks
explanation: Last night sets up past time. Walks should change to walked if both actions happened then.
hint: Find the verb that does not match Last night.
```
