## Teaching Goal

Teach students to recognize shared folktale patterns while preserving story differences.

## Student Outcome

The student can identify a pattern such as repeated test, magical help, youngest child, impossible task, or reward/consequence in a tale.

## Misconception Checks

Assuming all tales with the same pattern teach the same lesson; flattening cultural or story differences; naming a pattern without evidence.

## Questions

```question
key: u02_l04_q01_pattern_match
type: match-pairs
prompt: Match each folktale pattern to the event that shows it.
pairs:
  - left: Repeated test
    right: Three travelers each answer the same riddle.
  - left: Magical help
    right: A talking fish shows the hidden bridge.
  - left: Youngest child succeeds
    right: The third sister solves what the older two could not.
  - left: Reward after hardship
    right: The patient worker receives the seed that grows gold fruit.
explanation: A pattern must be shown by a story event.
hint: Pair the label with the event that actually displays it.
```

```question
key: u02_l04_q02_not_same_lesson
type: multiple-choice
prompt: Two tales both use magical help. Why might they still teach different lessons?
choices:
  - The helper can be used in different situations with different choices and consequences.
  - Magical help always teaches the same lesson.
  - A tale with magic cannot have a lesson.
  - The lesson depends only on how old the tale is.
correctAnswer: The helper can be used in different situations with different choices and consequences.
explanation: A shared pattern does not erase the details that shape each tale's lesson.
hint: Same pattern, different story details.
```

```question
key: u02_l04_q03_pattern_evidence
type: passage-question
prompt: Read the summary and identify the pattern.
passageTitle: The Three Tasks
passage: |
  The queen gave the gardener three tasks.
  First he sorted seeds by moonlight. Then he crossed a river without a boat.
  Last he found a bird that sang only at dawn.
question: Which folktale pattern is strongest in this summary?
choices:
  - Three repeated or rising tasks
  - A poem with stanzas
  - A city law being written
  - A narrator explaining history
correctAnswer: Three repeated or rising tasks
explanation: The gardener must complete three tasks, a common folktale pattern.
hint: Notice what happens more than once.
```

```question
key: u02_l04_q04_pattern_function
type: multiple-choice
prompt: In a tale, each of three siblings meets the same hungry stranger. What does this repeated pattern help the reader notice?
choices:
  - How each sibling responds differently to the same test.
  - That all siblings have the same name.
  - That strangers never matter in stories.
  - That the setting changes to a new country each time.
correctAnswer: How each sibling responds differently to the same test.
explanation: A repeated test makes the characters' different choices easier to compare.
hint: Ask what the repeated event reveals.
```

```question
key: u02_l04_q05_story_specific
type: passage-question
prompt: Read the summary and answer.
passageTitle: The Hidden Road
passage: |
  A girl receives a feather from a crane after freeing it from a snare.
  When she is lost, the feather points toward the hidden road home.
question: Which statement uses the pattern but keeps this tale's details?
choices:
  - Magical help comes after the girl shows mercy to the crane.
  - Every magical helper is a crane.
  - All lost children receive feathers.
  - The story teaches that snares are useful gifts.
correctAnswer: Magical help comes after the girl shows mercy to the crane.
explanation: This answer names the pattern and the specific act that leads to help.
hint: Keep both the pattern and this story's evidence.
```
