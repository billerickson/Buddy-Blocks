## Teaching Goal

Compare two retellings or versions by one named feature.

## Student Outcome

The student can identify a shared pattern, changed detail, and effect of the change.

## Misconception Checks

Treating difference as error; listing all differences; comparing without naming a focus.

## Questions

```question
key: u06_l01_q01_focus_version
type: multiple-choice
prompt: Which comparison focus is useful for two versions of the same tale?
choices:
  - How the helper changes the ending
  - Which version has more commas
  - Which title has fewer letters
  - How thick the book might be
correctAnswer: How the helper changes the ending
explanation: A helper's role can affect the story's meaning, so it is a useful focus.
hint: Choose a focus connected to story meaning.
```

```question
key: u06_l01_q02_same_pattern_changed_detail
type: passage-question
prompt: Read the two versions and answer.
passageTitle: The Hidden Door
passage: |
  Version A: A child opens the hidden door after sharing bread with a stranger.
  Version B: A child opens the hidden door after answering a stranger's riddle.
question: Which comparison is accurate?
choices:
  - Both versions have a stranger who helps lead to the hidden door, but the child's action changes.
  - Both versions say the child shares bread.
  - Both versions say the child answers a riddle.
  - Neither version includes a hidden door.
correctAnswer: Both versions have a stranger who helps lead to the hidden door, but the child's action changes.
explanation: The shared pattern is stranger plus hidden door; the changed detail is the child's action.
hint: Look for one same element and one changed element.
```

```question
key: u06_l01_q03_effect_of_change
type: passage-question
prompt: Read the two endings and choose the effect of the change.
passageTitle: Two Endings
passage: |
  Version A: The hero returns alone with the prize and receives all the praise.
  Version B: The hero returns with the helper and shares the praise before the city.
question: How does Version B change the story's meaning?
choices:
  - It gives more importance to gratitude and shared success.
  - It removes the helper from the ending.
  - It makes the quest fail.
  - It says praise is impossible.
correctAnswer: It gives more importance to gratitude and shared success.
explanation: Version B includes the helper in the return and praise.
hint: Ask what changed about the helper at the end.
```

```question
    right: One version uses a river, while the other uses a locked gate.

```
