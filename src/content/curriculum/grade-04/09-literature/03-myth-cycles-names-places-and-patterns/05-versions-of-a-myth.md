## Teaching Goal

Introduce version awareness by comparing two adapted myth retellings with respect.

## Student Outcome

The student can identify what stays similar and what changes between versions without calling every difference an error.

## Misconception Checks

Thinking one version must be the only correct story; comparing every difference; ignoring the shared pattern.

## Questions

```question
key: u03_l05_q01_version_idea
type: multiple-choice
prompt: What is a good way to think about two traditional versions of a myth?
choices:
  - They may share a pattern while changing some details.
  - One must be fake because details differ.
  - Differences mean the story has no meaning.
  - Only the longest version can be read.
correctAnswer: They may share a pattern while changing some details.
explanation: Traditional myths often have multiple retellings with shared patterns and changed details.
hint: Think same pattern, different details.
```

```question
key: u03_l05_q02_same_changed
type: passage-question
prompt: Read the two versions and answer.
passageTitle: Two Versions Of The Hidden Spring
passage: |
  Version A: A girl shares water with an old traveler. The traveler shows her the hidden spring.
  Version B: A boy shares bread with an old traveler. The traveler shows him the hidden spring.
question: What stays the same in both versions?
choices:
  - A child helps an old traveler and is shown the hidden spring.
  - The child always shares water.
  - The helper is a bird.
  - The spring is never found.
correctAnswer: A child helps an old traveler and is shown the hidden spring.
explanation: The shared pattern is helping the traveler and receiving guidance to the spring.
hint: Look for the pattern that appears in both summaries.
```

```question
key: u03_l05_q03_changed_detail_effect
type: passage-question
prompt: Read the versions and choose the effect of the change.
passageTitle: Two Endings
passage: |
  Version A: The proud hunter apologizes and is allowed to leave the sacred grove.
  Version B: The proud hunter refuses to apologize and must guard the grove for a year.
question: How does the changed ending affect the story?
choices:
  - Version A emphasizes repentance, while Version B emphasizes consequence.
  - Both versions have exactly the same lesson.
  - Version B has no consequence.
  - Version A removes the hunter from the story before he acts.
correctAnswer: Version A emphasizes repentance, while Version B emphasizes consequence.
explanation: The different ending changes what the reader notices about the hunter's response.
hint: Compare what the hunter chooses at the end.
```

```question
key: u03_l05_q06_difference_not_error
type: multiple-choice
prompt: In one version, the helper is an old woman. In another, the helper is a talking bird. What is the best reader response?
choices:
  - Notice how each helper changes the story's feeling or meaning.
  - Decide the talking bird version cannot be a myth.
  - Ignore both helpers because helpers never matter.
  - Assume the old woman and bird are the same character in every detail.

explanation: A changed helper can affect the version without making one version automatically wrong.
```
