## Teaching Goal

Teach students to decode what common academic task verbs ask them to do.

## Student Outcome

The student can match verbs such as analyze, infer, evaluate, summarize, cite, and support to the expected action.

## Misconception Checks

Analyze means answer quickly; summarize means copy the first sentence; evaluate means state an opinion only.

## Teaching Note

Put three prompts side by side and underline the verb that changes the task.

## Questions

```question
key: u03_l01_q01_verb_action_match
type: match-pairs
prompt: Match each academic verb to the action it asks for.
pairs:
  - left: summarize
    right: tell the main ideas briefly
  - left: cite
    right: point to exact evidence or a source
  - left: evaluate
    right: judge using criteria or evidence
  - left: analyze
    right: break into parts to understand how it works
explanation: Academic verbs tell what kind of thinking or response a task requires.
hint: Ask what the teacher expects the student to do.
```

```question
key: u03_l01_q02_prompt_action
type: multiple-choice
prompt: A prompt says, 'Analyze how the character changes after the storm.' What should the student do?
choices:
  - Break the change into parts and explain how it happens
  - Copy the first sentence of the story
  - List every character in the story
  - Give a one-word opinion about the storm
correctAnswer: Break the change into parts and explain how it happens
explanation: Analyze asks for parts and relationships, not a quick answer or copied sentence.
hint: Analyze usually asks how or why something works.
```

```question
key: u03_l01_q03_summarize_not_copy
type: multiple-choice
prompt: Which response best follows the direction 'Summarize the paragraph'?
choices:
  - The paragraph explains that bees help plants by moving pollen.
  - The first sentence says, 'Bees fly from flower to flower.'
  - I liked the paragraph because bees are interesting.
  - Bees, flowers, pollen, nectar, garden.
correctAnswer: The paragraph explains that bees help plants by moving pollen.
explanation: A summary gives the main idea briefly in your own words.
hint: A summary is short, complete, and focused on the main idea.
```

```question
key: u03_l01_q04_direction_cloze
type: fill-blank
prompt: Choose the verb that fits the task.
sentenceBefore: To
sentenceAfter: a claim, use evidence that proves or backs it up.
choices:
  - support
  - decorate
  - estimate
  - sequence
correctAnswer: support
explanation: Support means back up a claim with evidence or reasons.
hint: Which word means to back up an idea?
```

```question
key: u03_l01_q05_dialogue_response
type: dialogue-builder
prompt: Choose the best student response to the direction.
turns:
  - speaker: Teacher
    line: Evaluate which source is more useful for a report about local weather.
  - speaker: Student
    line: So I should...
choices:
  - judge the sources using evidence, such as date, author, and topic
  - read only the title of the first source
  - copy both sources exactly
  - choose the source with the most colorful picture
correctAnswer: judge the sources using evidence, such as date, author, and topic
explanation: Evaluate means judge carefully using criteria or evidence.
hint: Evaluate asks for a judgment, but not a random opinion.
```

```question
key: u03_l01_q06_explain_verb
type: multiple-choice
prompt: Which statement best explains the difference between identify and explain in school directions?
choices:
  - Identify means name or point out; explain means tell how or why.
  - Identify means write a long reason; explain means name only.
  - Both words mean copy one sentence from the text.
  - Both words mean give a personal opinion without evidence.
correctAnswer: Identify means name or point out; explain means tell how or why.
explanation: "Identify is usually shorter: name it. Explain requires reasoning, details, or how/why."
hint: One asks you to name; the other asks you to give reasoning.
```
