## Teaching Goal

Show how suffixes can change a word's role, meaning, and fit in a sentence.

## Student Outcome

The student can identify whether a suffix forms a noun, adjective, adverb, or action-related word in context.

## Misconception Checks

Suffixes only make words longer; words in the same family can be swapped without changing grammar; `-ly` always makes a word informal.

## Teaching Note

Compare create, creation, creative, and creatively in four short sentences.

## Questions

```question
key: u02_l02_q01_suffix_function_match
type: match-pairs
prompt: Match each suffix to a common job it can do.
pairs:
  - left: -tion
    right: forms a noun for an action or result
  - left: -ous
    right: forms an adjective meaning full of or having
  - left: -ly
    right: often forms an adverb showing how
  - left: -able
    right: forms an adjective meaning able to be
explanation: Suffixes can signal how a word works in a sentence.
hint: Think of words like action, dangerous, quickly, and readable.
```

```question
key: u02_l02_q02_word_form_cloze
type: multi-blank-cloze
prompt: Choose the form that fits each sentence.
parts:
  - "The artist will "
  - " a new design. Her "
  - " impressed the class."
blanks:
  - correctAnswer: create
    choices:
      - create
      - creation
      - creative
  - correctAnswer: creation
    choices:
      - create
      - creation
      - creatively
explanation: Create is the action verb. Creation is the noun that names what she made.
hint: Ask whether the sentence needs an action or a thing.
```

```question
key: u02_l02_q03_adverb_fit
type: multiple-choice
prompt: Which sentence uses a suffix to show how an action happened?
choices:
  - The speaker answered calmly.
  - The calm speaker answered.
  - The speaker's calm helped.
  - The room became calm.
correctAnswer: The speaker answered calmly.
explanation: Calmly uses -ly to describe how the speaker answered.
hint: Look for the word that describes an action.
```

```question
key: u02_l02_q04_correct_word_form
type: multiple-choice
prompt: Which correction uses a word form that makes sense?
choices:
  - The explanation was confusing to the class.
  - The explanation was confusion to the class.
  - The explanation was confuse to the class.
  - The explanation was confused the class.
correctAnswer: The explanation was confusing to the class.
explanation: The sentence needs an adjective such as confusing, or it must be rewritten to use the noun confusion.
hint: Ask whether the sentence needs a describing word or a noun.
```

```question
key: u02_l02_q05_suffix_meaning_able
type: fill-blank
prompt: Choose the word that means 'able to be read.'
sentenceBefore: A clear paragraph is
sentenceAfter: .
choices:
  - readable
  - reader
  - reading
  - readily
correctAnswer: readable
explanation: The suffix -able can mean able to be, so readable means able to be read.
hint: Which choice uses the suffix that means able to be?
```

```question
key: u02_l02_q06_explain_suffix_shift
type: multiple-choice
prompt: "Which explanation best shows the difference between active and activity in these sentences: 'The active puppy ran in circles. The activity lasted ten minutes'?"
choices:
  - Active describes the puppy, while activity is a noun that names what happened.
  - Active and activity can be switched because they share the same base.
  - Active is a noun, while activity describes how the puppy ran.
  - Activity means the puppy was not moving.
correctAnswer: Active describes the puppy, while activity is a noun that names what happened.
explanation: "The suffixes help show different roles: active describes, while activity names an action or event."
hint: Ask which word describes a noun and which word names a thing.
```
