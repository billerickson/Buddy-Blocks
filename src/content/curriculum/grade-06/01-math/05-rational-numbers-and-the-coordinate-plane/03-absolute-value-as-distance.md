## Teaching Goal

Interpret absolute value as distance from zero.

## Student Outcome

The student can find and explain absolute value in numerical and contextual situations.

## Key Ideas

Absolute value is never negative because distance is nonnegative, but context may still need direction.

## Misconception Checks

Absolute value means change the sign; `| -8 |` and `-8` mean the same thing; distance includes direction.

## Teaching Note

Ask how far -6 and +6 are from zero, then compare that distance with their positions.
