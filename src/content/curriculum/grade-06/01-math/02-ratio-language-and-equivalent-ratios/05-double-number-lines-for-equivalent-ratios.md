## Teaching Goal

Connect equivalent ratios to aligned positions on double number lines.

## Student Outcome

The student can use a double number line to find or interpret equivalent ratio pairs.

## Key Ideas

Aligned marks show quantities that grow together by the same scale factor.

## Misconception Checks

Marks can be uneven without meaning; aligned quantities can be mixed across different scale factors; only one side needs a pattern.

## Teaching Note

Build a double number line from a ratio table so students see the table rows become aligned marks.

## Questions

```question
key: u02_l05_q01_aligned_marks
type: fill-blank
prompt: A double number line shows 3 notebooks cost $12.
sentenceBefore: At 1 notebook, the aligned cost should be $
sentenceAfter: .
choices:
  - "3"
  - "4"
  - "9"
  - "12"
correctAnswer: "4"
explanation: $12 divided by 3 notebooks is $4 per notebook.
hint: Move from 3 notebooks to 1 notebook by dividing by 3.
```
