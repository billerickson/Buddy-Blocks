## Teaching Goal

Generate equivalent ratios using multiplicative scaling in tables.

## Student Outcome

The student can complete a ratio table by multiplying or dividing both quantities by the same factor.

## Key Ideas

Equivalent ratios preserve the same multiplicative relationship.

## Misconception Checks

Add the same number to both quantities; scale only one quantity; match rows by total rather than relationship.

## Teaching Note

Show a table row of 2 cups to 5 cups and ask what happens if the recipe doubles and triples.

## Questions

```question
key: u02_l04_q01_equivalent_ratio
type: multiple-choice
prompt: Which ratio is equivalent to 3:5?
choices:
  - 6:10
  - 4:6
  - 8:10
  - 6:8
correctAnswer: 6:10
explanation: 3 and 5 are both multiplied by 2 to get 6:10.
hint: Look for the same multiplication factor on both numbers.
```

```question
key: u02_l04_q03_scale_factor
type: fill-blank
prompt: Complete the scaling statement.
sentenceBefore: To go from 4:9 to 20:45, multiply both quantities by
sentenceAfter: .
choices:
  - "4"
  - "5"
  - "9"
  - "16"
correctAnswer: "5"
explanation: 4 x 5 = 20 and 9 x 5 = 45.
hint: Find what 4 was multiplied by to make 20.
```

```question
key: u02_l04_q04_not_equivalent
type: multiple-choice
prompt: Which ratio is NOT equivalent to 2:7?
choices:
  - 4:14
  - 6:21
  - 8:28
  - 10:21
correctAnswer: 10:21
explanation: 2 x 5 = 10, but 7 x 5 = 35, not 21.
hint: Check whether both numbers use the same multiplier.
```

