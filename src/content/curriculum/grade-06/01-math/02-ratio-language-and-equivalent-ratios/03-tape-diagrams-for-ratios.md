## Teaching Goal

Use tape diagrams to represent ratio units visually.

## Student Outcome

The student can interpret or complete a tape diagram for a ratio and use it to find related quantities.

## Key Ideas

Equal-size tape units show the repeated parts in a ratio relationship.

## Misconception Checks

Tape segments in one ratio must show actual object size; unequal segments can represent equal ratio units; totals are found by adding only one side.

## Teaching Note

Draw 2 equal parts for one quantity and 3 equal parts for another, then scale each part to a known value.

## Questions

```question
key: u02_l03_q01_tape_units
type: multiple-choice
prompt: A tape diagram for the ratio red:blue = 2:3 has 10 red marbles. How many marbles does one tape unit represent?
choices:
  - "2"
  - "3"
  - "5"
  - "10"
correctAnswer: "5"
explanation: The 2 red tape units represent 10 red marbles, so each unit is 5 marbles.
hint: Divide the red amount by the number of red tape units.
```

```question
key: u02_l03_q03_total_from_tape
type: fill-blank
prompt: A ratio tape has 2 parts juice and 5 parts water. Each part is 3 cups.
sentenceBefore: The total mixture is
sentenceAfter: cups.
choices:
  - "7"
  - "10"
  - "15"
  - "21"
correctAnswer: "21"
explanation: There are 2 + 5 = 7 tape parts, and 7 x 3 = 21 cups.
hint: Find the total number of equal parts first.
```
