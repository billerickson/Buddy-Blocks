## Teaching Goal

Help students apply memorized sayings to fitting situations.

## Student Outcome

The student can choose which saying fits a short scenario and explain the clue.

## Misconception Checks

Matching by one surface word; choosing a saying because it sounds familiar; ignoring the situation's main problem.

## Teaching Note

Read a two-sentence situation, ask what the problem is, then choose the saying that names the lesson.

## Questions

```question
key: u02_l03_q01_good_start_scenario
type: passage-question
prompt: Read the situation and choose the saying.
passageTitle: The First Page
passage: |
  Lina writes a clear outline before drafting her report.
  The rest of the writing goes smoothly.
question: Which saying best fits?
choices:
  - Well begun is half done.
  - A stitch in time saves nine.
  - Many hands make light work.
  - Measure twice, cut once.
correctAnswer: Well begun is half done.
explanation: Lina's strong beginning makes the rest of the work easier.
hint: Look for the saying about a good start.
```


```question
key: u02_l03_q05_explain_fit
type: multiple-choice
prompt: A student checks every answer before turning in a quiz. Which saying and reason fit?
choices:
  - Measure twice, cut once, because the student checks carefully before finishing.
  - Many hands make light work, because the student is working with a team.
  - A stitch in time saves nine, because the quiz has a tear in it.
  - Well begun is half done, because the student is starting a project.
correctAnswer: Measure twice, cut once, because the student checks carefully before finishing.
explanation: The key idea is careful checking before a final action.
hint: Which saying is about checking first?
```

```question
key: u02_l03_q06_bad_match
type: multiple-choice
prompt: Which match does not fit?
choices:
  - "A class cleans faster with helpers: Many hands make light work."
  - "A good outline helps a report: Well begun is half done."
  - "A loose shoelace is tied before someone trips: A stitch in time saves nine."
  - "A team wins because it runs fast: Measure twice, cut once."
correctAnswer: "A team wins because it runs fast: Measure twice, cut once."
explanation: Running fast does not show careful checking before action.
hint: Find the situation that does not show the saying's main idea.
```
