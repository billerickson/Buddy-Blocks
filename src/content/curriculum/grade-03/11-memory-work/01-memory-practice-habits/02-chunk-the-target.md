## Teaching Goal

Teach students to break a memory target into meaningful chunks before practicing.

## Student Outcome

The student can divide a short target into useful chunks and explain why the chunks help.

## Misconception Checks

Splitting a target at random; making chunks too large; thinking chunking changes the target.

## Teaching Note

Use "Look, chunk, cover, recall, check" and place each action on a separate card. Then combine the cards back into the full routine.

## Questions

```question
key: u01_l02_q01_best_chunks
type: multiple-choice
prompt: "Which chunking plan best fits this target: 'Look, chunk, cover, recall, check'?"
choices:
  - Look / chunk / cover / recall / check
  - Lo / okch / unkco / verrecallcheck
  - Look chunk cover recall check all at once
  - Look, chunk / cover, recall, check, remember, repeat
correctAnswer: Look / chunk / cover / recall / check
explanation: Each word is one clear action in the routine, so each can be a useful chunk.
hint: Good chunks keep meaningful parts together.
```

```question
key: u01_l02_q02_order_chunks
type: order-items
prompt: Put the memory routine chunks in order.
items:
  - check
  - look
  - recall
  - cover
  - chunk
correctOrder:
  - look
  - chunk
  - cover
  - recall
  - check
explanation: First see the model, then break it into parts, cover it, recall it, and check it.
hint: The routine starts before the target is covered.
```

```question
key: u01_l02_q03_missing_action

sentenceAfter: recall, check.

  - cover
  - race
  - erase
  - skip

hint: The missing step keeps you from looking at the answer.
```

```question

```
