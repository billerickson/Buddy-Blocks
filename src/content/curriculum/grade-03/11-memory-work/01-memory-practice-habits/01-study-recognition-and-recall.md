## Teaching Goal

Give students immediate success with one short memory line before naming any technical memory words.

## Student Outcome

The student can look at a short line, cover it, say it, and check whether the words are right.

## Misconception Checks

Thinking memory practice means only looking again; thinking a missed word means starting over; thinking the goal is speed instead of correct words.

## Teaching Note

Start with the line "North, east, south, west." Say it together, cover it, say it again, then uncover it and check. Use only the words look, cover, say, and check.
