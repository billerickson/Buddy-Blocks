## Teaching Goal

Assess cumulative recall, explanation, correction, and transfer across Memory Works 1.

## Student Outcome

The student can complete a mixed set and explain how to practice a new short target.

## Misconception Checks

Mastery means no mistakes; only exact-word targets count as memory work; a new target should be memorized by rereading only.

## Teaching Note

Present the challenge as a celebration of tools learned, not a high-stakes performance.

## Questions

```question
key: u06_l04_q01_new_target_strategy
type: multiple-choice
prompt: You need to memorize a new short line. What should you do first?
choices:
  - Understand it and break it into chunks.
  - Read it once and never check it.
  - Race through it as fast as possible.
  - Skip meaning and only count the words.
correctAnswer: Understand it and break it into chunks.
explanation: Meaning and chunking make a new memory target easier to practice well.
hint: Use the routine from the beginning of the level.
```

prompt: Which practice plan fits the new target 'Observe carefully before you conclude'?
