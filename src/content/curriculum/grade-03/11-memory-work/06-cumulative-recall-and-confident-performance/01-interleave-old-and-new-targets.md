## Teaching Goal

Teach students to practice mixed memory targets instead of only one kind at a time.

## Student Outcome

The student can identify target type, choose a strategy, and answer mixed recall prompts from earlier units.

## Misconception Checks

Mixed review is unfair; the newest target is the only one worth practicing; every prompt should use the same cue.

## Teaching Note

Present a mixed mini-deck and ask students to name the target type before answering.

## Questions

```question
key: u06_l01_q01_target_strategy_pairs
type: match-pairs
prompt: Match each target to a good recall strategy.
pairs:
  - left: poem line
    right: use a first-word cue
  - left: measure fact
    right: check the units
  - left: timeline peg
    right: use before-and-after order
  - left: saying
    right: use the meaning
explanation: Interleaving works best when you choose the strategy that fits the target.
hint: Ask what kind of memory target each item is.
```
