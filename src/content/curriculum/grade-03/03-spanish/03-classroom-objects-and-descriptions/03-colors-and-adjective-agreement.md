## Teaching Goal

Teach basic object descriptions with color words that follow and agree with nouns.

## Student Outcome

The student can describe familiar classroom objects with color adjectives in correct position and basic agreement.

## Misconception Checks

Putting every adjective before the noun; using one color form for every noun; changing invariant colors unnecessarily.

## Teaching Note

Compare one masculine and one feminine object with the same color so students see the agreement cue.

## Questions

```question
key: u03_l03_q01_carpeta_roja
type: multiple-choice
prompt: Choose the phrase that means the red folder.
choices:
  - la carpeta roja
  - la carpeta rojo
  - roja la carpeta
  - el carpeta rojo
correctAnswer: la carpeta roja
explanation: Carpeta is feminine, so the color form is roja, and the adjective follows the noun.
hint: Check both word order and the adjective ending.
```

```question
key: u03_l03_q02_order_book_red
type: order-items
prompt: "Put the Spanish phrase in order: the red book."
items:
  - rojo
  - libro
  - el
correctOrder:
  - el
  - libro
  - rojo
explanation: "Many Spanish color adjectives follow the noun: el libro rojo."
hint: Start with the article and noun, then the color.
```

```question
key: u03_l03_q04_fix_color

acceptedAnswers:
  - la mesa roja

```

