## Teaching Goal

Teach respectful praise and blame as evaluation of actions, choices, and evidence.

## Student Outcome

The student can revise a personal attack or exaggeration into a fair, action-focused evaluation.

## Misconception Checks

Praise means exaggeration; blame means insult; respectful wording makes evaluation weak.

## Teaching Note

Compare "He is terrible" with "His choice was unfair because he changed the rule after the game started."

## Questions

```question
key: u05_l04_q01_respectful_praise
type: multiple-choice
prompt: Which sentence gives respectful, specific praise?
choices:
  - Your explanation was clear because you named each step.
  - Your explanation was good.
  - You are smart at explaining.
  - That was good, I guess.
correctAnswer: Your explanation was clear because you named each step.
explanation: The praise names the action and gives evidence.
hint: Look for praise that is specific and action-focused.
```

```question
key: u05_l04_q02_revise_attack
type: error-correction
prompt: "Revise this unfair comment so it focuses on the action: 'Sam is awful because he grabbed all the markers.'"
sentence: Sam is awful because he grabbed all the markers.
acceptedAnswers:
  - Sam's choice was unfair because he grabbed all the markers.
  - It was unfair for Sam to grab all the markers.
  - Grabbing all the markers was unfair because others could not use them.
explanation: The revision evaluates the action instead of attacking Sam as a person.
hint: Change the sentence so it judges the choice, not the person.
```

```question
key: u05_l04_q03_fair_response_dialogue
type: multiple-choice
prompt: Maya says, "I took all the glue sticks so my group could finish first." Which reply from Jules gives fair criticism without insulting Maya?

  - That choice was unfair because other groups needed glue too.
  - That was a selfish thing to do.
  - Our group needed glue too.
  - Maybe other groups should wait.

explanation: The response names the action, gives evidence, and avoids personal attack.
hint: Choose the line that evaluates the choice respectfully.
```

```question

  Comment A: Your reason was helpful because it used an example.
  Comment B: You are bad at explaining.
question: Which comment is a personal attack?
choices:
  - Comment B
  - Comment A
  - Both comments
  - Neither comment
correctAnswer: Comment B
explanation: Comment B attacks the person instead of focusing on a specific action or choice.
hint: Look for the comment that labels the person.
```

```question
key: u05_l04_q05_unfair_to_fair
type: match-pairs
prompt: Match each unfair comment to a fairer version.
pairs:
  - left: You are careless.
    right: The poster needs checking because two names are missing.
  - left: Your idea is dumb.
    right: The idea needs a clearer reason so the audience understands.
  - left: You never help.
    right: This time, the group needed you to take one job.
  - left: That speech was bad.
    right: The speech would be clearer with a slower pace.
explanation: Fair evaluation focuses on the action and how to improve it.
hint: Match each attack with the version that names a fixable action.
```
