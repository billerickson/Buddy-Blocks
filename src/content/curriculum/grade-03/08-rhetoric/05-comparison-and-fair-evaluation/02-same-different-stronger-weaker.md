## Teaching Goal

Build accurate comparison and contrast language connected to evidence.

## Student Outcome

The student can use same, different, stronger, and weaker to compare two actions or ideas with support.

## Misconception Checks

Stronger means louder or more exciting; different means one is bad; comparison words do not need evidence.

## Teaching Note

Use a comparison chart with one criterion and evidence under each option.

## Questions

```question
key: u05_l02_q01_comparison_word
type: fill-blank
prompt: Choose the best word.
sentenceBefore: Both plans include a cleanup job, so they are the
sentenceAfter: in that way.
choices:
  - same
  - different
  - stronger
  - weaker
correctAnswer: same
explanation: The plans share a feature, so they are the same in that way.
hint: Look for what both plans have.
```

```question
key: u05_l02_q02_stronger_with_evidence
type: multiple-choice
prompt: "Criterion: clarity. Which comparison is best supported?"
choices:
  - Plan A is clearer because it numbers the steps.
  - Plan A is stronger because I like it.
  - Plan B is weaker because I do not like its title.
  - Both plans are different because they exist.
correctAnswer: Plan A is clearer because it numbers the steps.
explanation: Numbered steps are evidence about clarity.
hint: Choose the comparison that uses the criterion and evidence.
```

```question
key: u05_l02_q03_language_to_evidence
type: match-pairs
prompt: Match each comparison phrase to the evidence.
pairs:
  - left: same
    right: Both responses include a reason.
  - left: different
    right: One response has an example, and the other does not.
  - left: stronger
    right: This reason connects directly to the claim.
  - left: weaker
    right: This detail changes the topic.
explanation: Comparison words should be tied to evidence.
hint: Match the word to the clue that proves it.
```

```question
key: u05_l02_q04_compare_responses
type: passage-question
prompt: Read the two responses and answer.
passageTitle: Two Reasons
passage: |
  Response A: We should practice because some students missed the last rehearsal.
  Response B: We should practice because the wall is white.
question: Which response has the stronger reason?
choices:
  - Response A, because it explains why practice is needed.
  - Response B, because walls can be white.
  - Both are equally strong because both use because.
  - Neither has a claim.
correctAnswer: Response A, because it explains why practice is needed.
explanation: Response A's reason is relevant to the claim about practicing.
hint: Ask which reason supports the claim.
```

```question
key: u05_l02_q05_write_comparison
type: multiple-choice
prompt: "Criterion: specific support. A says, 'It helps.' B says, 'It helps students find scissors quickly.' Which comparison is best?"
choices:
  - B has stronger support because it gives a specific example about finding scissors.
  - A has stronger support because it is shorter.
  - Both are equally specific because both mention helping.
  - B is weaker because scissors are classroom supplies.
correctAnswer: B has stronger support because it gives a specific example about finding scissors.
explanation: Specific support is stronger when it shows exactly how or why.
hint: Look for the response with a clear example.
```

```question
key: u05_l02_q06_unsupported_comparison
type: multiple-choice
prompt: Which comparison is unsupported?
choices:
  - Plan B is better.
  - Plan A is clearer because it has numbered steps.
  - Plan B is fairer because everyone gets a turn.
  - Plan A is more useful because it lists supplies.
correctAnswer: Plan B is better.
explanation: It gives no criterion or evidence.
hint: A supported comparison says why.
```
