## Teaching Goal

Teach students that a comparison needs one clear basis or criterion.

## Student Outcome

The student can choose a criterion that fits two things being compared.

## Misconception Checks

Comparing means naming a favorite; the criterion can change during the comparison; any difference is equally important.

## Teaching Note

Compare two classroom jobs by one criterion at a time, such as helpfulness or difficulty.

## Questions

```question
key: u05_l01_q01_best_criterion
type: multiple-choice
prompt: Two groups made cleanup plans. Which criterion would help compare the plans?
choices:
  - Which plan is clearer?
  - Which plan has more letters in its title?
  - Which plan is on whiter paper?
  - Which plan was written closer to the window?
correctAnswer: Which plan is clearer?
explanation: Clarity is a useful criterion for comparing plans.
hint: A criterion should help judge what matters about the choices.
```

```question
key: u05_l01_q02_criteria_to_questions
type: match-pairs
prompt: Match each criterion to the comparison question.
pairs:
  - left: fairness
    right: Which choice gives everyone a turn?
  - left: usefulness
    right: Which tool helps finish the job?
  - left: clarity
    right: Which explanation is easier to understand?
  - left: carefulness
    right: Which action avoids mistakes?
explanation: A criterion names what feature or quality you are comparing.
hint: Match the criterion to the question that asks about that quality.
```

```question
key: u05_l01_q03_criterion_in_context
type: passage-question
prompt: Read the situation and answer.
passageTitle: Choosing Directions
passage: |
  Two students wrote directions for setting up a game.
  One set has numbered steps.
  The other set has all steps in one long sentence.
question: Which criterion fits this comparison best?
choices:
  - clarity
  - neat handwriting
  - paper color
  - longest title
correctAnswer: clarity
explanation: The comparison is about which directions are easier to understand.
hint: Ask what quality matters for directions.
```

```question
key: u05_l01_q05_name_criterion
type: multiple-choice
prompt: Two classmates suggest ways to share one tablet. Which criterion best compares whether the ideas are fair?

  - whether everyone gets equal time
  - which idea uses the longest words
  - which student sits closer to the window
  - whether the tablet case is blue
```
