## Teaching Goal

Distinguish meaning-focused revision from surface editing.

## Student Outcome

The student can identify whether a change improves meaning, order, support, word choice, or mechanics.

## Misconception Checks

Revision means spelling only; any change is revision; a draft is finished once mechanics are correct.

## Teaching Note

Show two changes to the same sentence: one fixes a capital letter and one adds a needed reason.

## Questions

```question
key: u06_l01_q01_revision_or_editing
type: multiple-choice
prompt: Which change is revision, not just editing?
choices:
  - Adding a reason that explains the claim
  - Fixing a missing capital letter
  - Correcting a spelling mistake
  - Adding a period at the end
correctAnswer: Adding a reason that explains the claim
explanation: Revision improves the meaning or effect of the writing. Editing fixes surface correctness.
hint: Look for the change that helps the message itself.
```

```question
key: u06_l01_q02_change_to_target
type: match-pairs
prompt: Match each change to what it improves.
pairs:
  - left: Add an example after the reason.
    right: support
  - left: Move the ending after the main action.
    right: order
  - left: Replace 'nice' with 'patient.'
    right: word choice
  - left: Capitalize the first word.
    right: mechanics
explanation: Revision can improve support, order, and word choice. Editing often fixes mechanics.
hint: Ask what part of the writing changes.
```

```question
key: u06_l01_q03_revision_in_context
type: passage-question
prompt: Read the draft and answer.
passageTitle: Draft
passage: |
  The class should use a cleanup chart.
  It is good.
question: Which revision most improves the meaning?
choices:
  - "Add: 'because everyone would know their job.'"
  - Change class to Class.
  - Add a period after good.
  - Write the same two sentences again.
correctAnswer: "Add: 'because everyone would know their job.'"
explanation: The added reason gives support for the claim.
hint: Choose the change that helps the audience understand why.
```

```question
key: u06_l01_q05_explain_revision_choice
type: multiple-choice
prompt: A draft says, 'The library is helpful.' Which revision would make the message stronger?
choices:
  - "Add an example: 'It has books about our project topic.'"
  - Capitalize the word library.
  - Add a period after helpful.
  - Write the same sentence twice.

explanation: A useful revision can add a reason, example, or more precise detail that helps the audience understand the idea.

```

```question
key: u06_l01_q06_not_revision
type: multiple-choice
prompt: Which change is mostly editing?
choices:
  - Fixing 'teh' to 'the'
  - Adding a specific example
  - Moving the conclusion to the end
  - Replacing a vague reason
correctAnswer: Fixing 'teh' to 'the'
explanation: Fixing a spelling mistake is editing. It may be important, but it is not meaning-focused revision.
hint: Editing often fixes spelling, capitals, or punctuation.
```
