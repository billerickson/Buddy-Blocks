## Teaching Goal

Help students decide which Rhetoric 1 move fits a task: retell, describe, support, compare, revise, or present.

## Student Outcome

The student can choose a fitting rhetorical move for a short scenario and explain the clue.

## Misconception Checks

The newest skill fits every task; a task can be answered well without noticing its purpose; describing and supporting are the same move.

## Teaching Note

Present a move menu and sort five short prompts before answering any of them.

## Questions

```question
key: u07_l01_q01_task_to_move
type: match-pairs
prompt: Match each task to the rhetorical move it asks for.
pairs:
  - left: Tell the key events of a fable in order.
    right: retell
  - left: Add sensory details to a sentence.
    right: describe
  - left: Give a claim with a reason.
    right: support
  - left: Name how two plans are alike and different.
    right: compare
explanation: Different tasks call for different rhetorical moves.
hint: Look for clue words such as tell, add details, reason, or compare.
```

```question
key: u07_l01_q02_best_move
type: multiple-choice
prompt: "Prompt: 'Which cleanup plan is fairer? Use one reason.' Which rhetorical move fits best?"
choices:
  - compare
  - retell
  - describe
  - revise
correctAnswer: compare
explanation: The prompt asks which plan is fairer, so it calls for comparison using a criterion and reason.
hint: The word 'fairer' is a comparison clue.
```

```question
key: u07_l01_q03_move_in_context
type: passage-question
prompt: Read the task and answer.
passageTitle: Studio Task
passage: |
  Read the short story.
  Tell what happened first, next, and last.
question: Which move does this task ask for?
choices:
  - retell
  - evaluate with evidence
  - revise word choice
  - choose a speaker
correctAnswer: retell
explanation: The task asks for events in order, which is retelling.
hint: First, next, and last are order clues.
```

```question
key: u07_l01_q04_prompt_clue
type: multiple-choice
prompt: "Prompt: 'Say your response aloud with a clear voice.' Which rhetorical move fits best?"
choices:
  - present
  - retell
  - compare
  - describe
correctAnswer: present
explanation: The prompt asks the student to deliver a response aloud, so it calls for presenting.
hint: Look for the clue about speaking to listeners.
```

```question
key: u07_l01_q05_explain_move_choice
type: multiple-choice
prompt: "Prompt: 'Make this request more respectful for a teacher.' Which move and clue fit best?"
choices:
  - Revise, because the prompt asks you to make the request more respectful.
  - Retell, because the prompt asks for events in order.
  - Compare, because the prompt asks which choice is fairer.
  - Present, because the prompt asks for a clear voice.
correctAnswer: Revise, because the prompt asks you to make the request more respectful.
explanation: The task asks for a change that improves audience fit, so it calls for revision.
hint: Look for words that ask you to improve or change the sentence.
```

```question
key: u07_l01_q06_describe_or_support
type: multiple-choice
prompt: Which prompt asks you to support, not just describe?
choices:
  - State which sign is clearer and give one reason.
  - Describe the color and shape of the sign.
  - List where the sign is hanging.
  - Add one sensory detail about the hallway.
correctAnswer: State which sign is clearer and give one reason.
explanation: Giving a reason supports a claim. The other prompts ask for description.
hint: Look for a claim plus why.
```
