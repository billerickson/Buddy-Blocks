## Teaching Goal

Diagnose what is weak or missing in a short response and choose a focused fix.

## Student Outcome

The student can identify whether a response needs audience fit, order, detail, support, comparison, or revision.

## Misconception Checks

Weak responses only have spelling problems; the fix is always to add more; all weak responses have the same problem.

## Teaching Note

Show three weak responses with different problems and name the missing rhetorical move in each.

## Questions

```question
key: u07_l02_q01_diagnose_weak_response
type: multiple-choice
prompt: "Weak response: 'The class should use a chart because charts are helpful.' What is the main problem?"
choices:
  - The reason is vague and does not explain how a chart helps.
  - The response has too many specific examples.
  - The response clearly names a comparison criterion.
  - The sentence retells story events in order.
correctAnswer: The reason is vague and does not explain how a chart helps.
explanation: The support says the chart is helpful, but it does not explain what the chart helps people do.
hint: Ask whether the reason answers why.
```

```question
key: u07_l02_q02_weakness_to_fix
type: match-pairs
prompt: Match each weakness to the best fix.
pairs:
  - left: No audience fit
    right: Make the wording respectful for the listener.
  - left: Events out of order
    right: Move the beginning before the outcome.
  - left: No example
    right: Add a specific case that shows the reason.
  - left: No comparison criterion
    right: Name the quality being compared.
explanation: A useful fix should match the actual weakness.
hint: Do not add more words until you know what is missing.
```

```question
key: u07_l02_q03_fix_vague_response
type: multiple-choice
prompt: Which revision adds a clearer reason?
choices:
  - The group should use a checklist because it helps everyone remember each step.
  - The group should use a checklist because it is nice.
  - The group should use a checklist because it is a checklist.
  - The group should use a checklist because it looks cool.
correctAnswer: The group should use a checklist because it helps everyone remember each step.
explanation: The revision gives a specific reason the checklist helps.
hint: Tell what the checklist helps the group do.
```

```question
key: u07_l02_q04_missing_audience_fit
type: passage-question
prompt: Read the response and answer.
passageTitle: Weak Request
passage: |
  Prompt: Ask the librarian for help finding a book.
  Response: Find it for me now.
question: Which fix would help most?
choices:
  - Make the request respectful and clear.
  - Add three color words.
  - Retell a fable first.
  - Compare two cleanup plans.
correctAnswer: Make the request respectful and clear.
explanation: The response misses audience fit for asking a librarian.
hint: Think about who hears the request.
```

```question
key: u07_l02_q05_diagnose_and_fix
type: multiple-choice
prompt: "Weak response: 'Plan A is better because I like it.' Which diagnosis and fix are best?"
choices:
  - "Weakness: no clear criterion or evidence. Fix: Plan A is clearer because it numbers the steps."
  - "Weakness: too much evidence. Fix: remove the reason."
  - "Weakness: too respectful. Fix: add a personal attack."
  - "Weakness: too many events. Fix: retell a fable."
correctAnswer: "Weakness: no clear criterion or evidence. Fix: Plan A is clearer because it numbers the steps."
explanation: A useful diagnosis names what is missing, then repairs that part with a criterion and evidence.
hint: Ask what 'better' means and what evidence is given.
```

```question
key: u07_l02_q06_more_not_always_better
type: multiple-choice
prompt: A response is weak because the reason is irrelevant. Which fix is best?
choices:
  - Replace the reason with one that supports the claim.
  - Add five unrelated details.
  - Make every word longer.
  - Write the same reason twice.
correctAnswer: Replace the reason with one that supports the claim.
explanation: The fix should repair relevance, not just make the response longer.
hint: A focused fix matches the weakness.
```
