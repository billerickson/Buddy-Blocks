## Teaching Goal

Connect multi-digit addition to place-value units and written methods.

## Student Outcome

The student can add within 1,000 using models, expanded form, or a written method and explain regrouping when it occurs.

## Key Ideas

Add hundreds with hundreds, tens with tens, and ones with ones; regroup when a place has enough for a larger unit.

## Misconception Checks

Adding digits without attending to place; forgetting regrouped tens or hundreds; treating the algorithm as unexplained steps.

## Teaching Note

Model 246 + 178 with expanded form before showing the compact written version.
