## Teaching Goal

Connect skip counting to equal jumps and equal groups rather than rote chanting alone.

## Student Outcome

The student can skip count by 2s, 5s, 10s, and 100s and identify the equal jump or group size.

## Key Ideas

Skip counting shows repeated equal increases on a number line or in equal groups.

## Misconception Checks

Memorizing a sequence without knowing the jump size; switching jump sizes midstream; counting the starting number as the first jump.

## Teaching Note

Show 0, 5, 10, 15 on a number line and label each jump as "+5."

## Questions

```question
key: u01_l06_q01_jump_size

prompt: What is the jump size in this sequence? 0, 5, 10, 15, 20
choices:
  - "+2"
  - "+5"
  - "+10"
  - "+15"
correctAnswer: "+5"
explanation: Each number is 5 more than the number before it.
hint: Compare two neighboring numbers.
```

```question
key: u01_l06_q02_complete_sequence
type: fill-blank
prompt: Complete the skip-counting sequence.
sentenceBefore: 12, 14, 16,
sentenceAfter: ", 20"
choices:
  - "17"
  - "18"
  - "19"
  - "22"
correctAnswer: "18"
explanation: The sequence counts by 2s, so the missing number is 18.
hint: Add the same jump each time.
```

```question
key: u01_l06_q03_groups_to_sequence
type: multiple-choice
prompt: There are 4 boxes with 10 pencils in each box. Which sequence counts the pencils by boxes?
choices:
  - 10, 20, 30, 40
  - 4, 8, 12, 16
  - 10, 11, 12, 13
  - 4, 14, 24, 34
correctAnswer: 10, 20, 30, 40
explanation: Each box adds 10 pencils, so the total counts by 10s.
hint: The jump size is the number of pencils in one box.
```
