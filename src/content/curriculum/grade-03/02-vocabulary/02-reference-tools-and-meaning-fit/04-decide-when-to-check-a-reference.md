## Teaching Goal

Teach students to decide when context is enough and when a reference check is useful.

## Student Outcome

The student can choose a strategy path for a word: use context, check a reference, or use both.

## Misconception Checks

Looking up every unknown word before thinking; refusing to check a reference when context is weak; treating reference use as a sign of failure.

## Teaching Note

Compare one sentence with a clear restatement clue and another where the context only hints at the meaning.

## Questions

```question
key: u02_l04_q01_clear_context
type: multiple-choice
prompt: 'Read: "The puppy was tiny, or very small." What should you do first to understand tiny?'
choices:
  - Use the clear context clue
  - Ignore the sentence and guess
  - Choose the first dictionary definition
  - Ask what puppy means
correctAnswer: Use the clear context clue
explanation: "The sentence gives a clear restatement: very small."
hint: Look for a clue that already explains the word.
```

```question
key: u02_l04_q02_weak_context
type: multiple-choice
prompt: 'Read: "The scientist studied the mineral." You are not sure what mineral means. What is the best next step?'
choices:
  - Check a glossary or dictionary
  - Pretend mineral means scientist
  - Skip the whole sentence
  - Use studied as the definition
correctAnswer: Check a glossary or dictionary
explanation: The sentence gives only a topic, not a clear meaning, so a reference check would help.
hint: Does the sentence explain the word, or only show where it appears?
```

```question
key: u02_l04_q03_strategy_order
type: order-items
prompt: Put these steps in a useful order when context is not enough.
items:
  - Choose the definition that fits
  - Make a first guess from context
  - Read a reference entry
  - Reread the sentence
correctOrder:
  - Reread the sentence
  - Make a first guess from context
  - Read a reference entry
  - Choose the definition that fits
explanation: Use context first, then use the reference to verify the meaning that fits.
hint: Start with the sentence before the reference.
```

```question
key: u02_l04_q04_context_plus_reference
type: passage-question
prompt: Read the sentence and entry.
passageTitle: Check Or Not?
passage: |-
  Sentence: The class will construct a model bridge from craft sticks.
  Entry: construct: to build or make something.
question: How does the entry help?
choices:
  - It confirms that construct means build or make
  - It changes the sentence topic to art
  - It tells the bridge is real
  - It shows construct means break
correctAnswer: It confirms that construct means build or make
explanation: The sentence hints at making a model, and the entry confirms the meaning.
hint: Compare model bridge from craft sticks with the entry definition.
```

```question
key: u02_l04_q05_strategy_match
type: match-pairs
prompt: Match each situation to the best first strategy.
pairs:
  - left: The sentence says "frigid, or very cold."
    right: Use the restatement clue
  - left: The word has two possible definitions in the entry.
    right: Choose the one that fits context
  - left: The sentence only says "the artifact was old."
    right: Check a reference for artifact
  - left: The word begins with re- and the sentence says it happened again.
    right: Use word part and context
explanation: Different word problems call for different strategies.
hint: Look at what evidence each situation gives you.
```
