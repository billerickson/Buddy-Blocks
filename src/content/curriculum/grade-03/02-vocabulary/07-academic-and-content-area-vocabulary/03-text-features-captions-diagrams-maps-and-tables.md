## Teaching Goal

Teach vocabulary for common text features and how those features help meaning.

## Student Outcome

The student can identify a caption, diagram, map, table, glossary, or timeline and explain what information it gives.

## Misconception Checks

Skipping text features; confusing table as furniture with table as a data display; treating captions as optional labels only.

## Teaching Note

Show a simple animal diagram with a caption and ask what each feature tells a reader.

## Questions

```question
key: u07_l03_q01_feature_match
type: match-pairs
prompt: Match each text feature to its purpose.
pairs:
  - left: caption
    right: explains a picture
  - left: diagram
    right: shows parts with labels
  - left: map
    right: shows places
  - left: table
    right: organizes information in rows and columns
explanation: Text features help readers understand information.
hint: Think about what each feature shows.
```

```question
key: u07_l03_q02_caption_meaning
type: multiple-choice
prompt: 'A photo of a turtle has this caption: "A hatchling crawls toward the ocean." What does the caption tell you?'
choices:
  - The turtle is a hatchling moving toward the ocean
  - The photo is about a mountain
  - The turtle is building a nest
  - Captions never give information
correctAnswer: The turtle is a hatchling moving toward the ocean
explanation: A caption explains what is shown in a picture.
hint: Use the words under or near the photo.
```

  - information arranged in rows and columns

  - a kind of map
