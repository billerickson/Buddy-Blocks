## Teaching Goal

Teach students to decide which vocabulary strategy should lead in a mixed situation.

## Student Outcome

The student can choose a strategy such as context clue, word part, reference check, relationship, or nonliteral interpretation and explain why it fits.

## Misconception Checks

Using the newest strategy for every word; choosing a reference check when context is clear; forcing a word-part clue that does not help.

## Teaching Note

Present three words: one with a clear context clue, one with a prefix, and one with multiple definitions, then compare the best first move.

## Questions

```question
key: u08_l01_q01_strategy_match
type: match-pairs
prompt: Match each word problem to the best first strategy.
pairs:
  - left: The sentence says "frigid, or very cold."
    right: use a restatement clue
  - left: The word is reread.
    right: use the prefix re-
  - left: The word has two dictionary definitions.
    right: choose the definition that fits
  - left: The phrase is "under the weather."
    right: think about idiom meaning
explanation: Different vocabulary problems call for different first strategies.
hint: Look at what kind of evidence is available.
```

```question
key: u08_l01_q02_best_strategy_context
type: multiple-choice
prompt: 'Read: "The kitten was timid; it hid when visitors arrived." What is the best first strategy for timid?'
choices:
  - Use the context clue
  - Look for a suffix
  - Treat it as an idiom
  - Choose the first dictionary definition without reading
correctAnswer: Use the context clue
explanation: "The sentence gives a clear clue: the kitten hid."
hint: Ask which strategy uses the evidence already in the sentence.
```

```question
key: u08_l01_q03_strategy_for_prefix
type: multiple-choice
prompt: Which strategy best helps with the word unhappy?
choices:
  - Use the prefix un- and the base word happy
  - Treat it as a homophone
  - Look for a map
  - Use only the first letter
correctAnswer: Use the prefix un- and the base word happy
explanation: Un- can mean not, so unhappy means not happy.
hint: Look for a meaningful word part at the beginning.
```

```question
key: u08_l01_q04_strategy_passage
type: passage-question
prompt: Read the map context and choose a strategy.
passageTitle: A Map Word
passage: The map key explained each symbol. A blue line meant river, and a star meant capital city.
question: Which strategy best helps you understand key?
choices:
  - Use the map context and meaning fit
  - Use the prefix k-
  - Treat key as a sound word
  - Choose the meaning "a metal object for a lock"
correctAnswer: Use the map context and meaning fit
explanation: Map context shows key means a guide to symbols, not a lock key.
hint: Use the words map and symbol.
```

```question
key: u08_l01_q05_strategy_order
type: order-items
prompt: Put the mixed word-solving steps in a useful order.
items:
  - Check if the meaning fits
  - Notice what kind of clue you have
  - Try a meaning
  - Reread the sentence
correctOrder:
  - Reread the sentence
  - Notice what kind of clue you have
  - Try a meaning
  - Check if the meaning fits
explanation: Even in mixed review, rereading and checking fit keep the strategy grounded.
hint: Start with the sentence and end by checking.
```

```question
key: u08_l01_q06_strategy_explain
type: multiple-choice
prompt: 'Read: "The careless painter spilled blue paint on the floor." Which strategy and reason fit careless?'
choices:
  - "Use word parts: care is the base word and -less means without."
  - "Use an idiom: careless is a phrase that cannot be solved."
  - "Use a homophone: careless sounds like another word."
  - "Use a map: careless shows a place."
correctAnswer: "Use word parts: care is the base word and -less means without."
explanation: Careless has a meaningful base and suffix, and the spilling clue confirms the meaning.
hint: Use the base word and suffix.
```
