## Teaching Goal

Teach students to distinguish broad curiosity from a question that can be answered with available sources.

## Student Outcome

The student can choose or improve a focused historical or civic inquiry question.

## Misconception Checks

Asking questions too broad for the source set; asking opinion questions when evidence is needed; assuming any question is equally useful.

## Teaching Note

Compare "What was life like long ago?" with "What does the photo show about how children traveled to school?"

## Questions

```question
key: u08_l01_q01_focused_question
type: multiple-choice
prompt: A photo shows children walking on a dirt road toward a small schoolhouse. Which inquiry question is most focused for this source?
choices:
  - What does the photo show about how children traveled to school?
  - What was everything about life long ago?
  - What will schools be like forever?
  - Which country is the best?
correctAnswer: What does the photo show about how children traveled to school?
explanation: The focused question can be answered with details in the photo.
hint: Choose the question that matches what the source can show.
```

```question
key: u08_l01_q02_answerable_question
type: passage-question
prompt: Use the source set to answer.
passageTitle: Small Source Set
passage: |
  Source 1: A map labels a river beside the town.
  Source 2: A photo shows a bridge across the river.
question: Which question can this source set answer best?
choices:
  - How did people cross the river near town?
  - What did every family eat for dinner?
  - What songs did children sing?
  - What will happen next century?
correctAnswer: How did people cross the river near town?
explanation: The map and photo both give evidence about the river crossing.
hint: Look for the question about the clues the sources actually provide.
```

```question
key: u08_l01_q03_fix_broad_question
type: multiple-choice
prompt: Which focused question can be answered by a map showing a town beside a river?
choices:
  - Where was the town located?
  - What was everything about the town's history?
  - Why did every family move to the town?
  - What foods did people eat in the town?
correctAnswer: Where was the town located?
explanation: A focused question should match the map evidence available.
hint: Ask about location or the river, because that is what the map shows.
```

```question
key: u08_l01_q04_focused_blank
type: fill-blank
prompt: Complete the sentence.
sentenceBefore: A focused inquiry question can be answered with the
sentenceAfter: provided.
choices:
  - sources
  - opinions
  - guesses
  - memories only
correctAnswer: sources
explanation: Inquiry questions should fit the evidence available.
hint: Focused questions need evidence.
```

```question
key: u08_l01_q05_broad_to_focused
type: match-pairs
prompt: Match each broad question to a more focused version.
pairs:
  - left: What was travel like?
    right: What route does the map show?
  - left: What did people do?
    right: What work does the photo show?
  - left: What happened long ago?
    right: Which event happened first on the timeline?
  - left: What did citizens value?
    right: Which ideal appears in the motto?
explanation: Focused questions name the kind of evidence needed.
hint: Look for the focused question that fits the same topic.
```

```question
key: u08_l01_q06_write_focused_question
type: multiple-choice
prompt: A timeline has three events about a bridge being built, repaired, and replaced. Which inquiry question is focused and answerable from this source?
choices:
  - Which bridge event happened first?
  - What was everything about town life?
  - Which bridge is the best in the world?
  - What will all bridges look like forever?
correctAnswer: Which bridge event happened first?
explanation: A focused timeline question should be answerable by reading the events in order.
hint: Use words like first, after, before, repaired, or replaced.
```
