## Teaching Goal

Teach that rights and responsibilities are connected.

## Student Outcome

The student can match a right with a related responsibility and explain respectful participation.

## Misconception Checks

Thinking rights mean doing anything one wants; treating responsibilities as punishments or chores only; ignoring other people's rights.

## Teaching Note

Use a discussion example: the right to speak connects with the responsibility to listen respectfully.

## Questions

```question
key: u04_l03_q01_right_responsibility_pairs
type: match-pairs
prompt: Match each right with a related responsibility.
pairs:
  - left: right to speak
    right: listen when others speak
  - left: right to use the library
    right: return shared books
  - left: right to be safe
    right: follow safety rules
  - left: right to learn
    right: let classmates learn too
explanation: Rights and responsibilities work together in community life.
hint: Ask what action helps protect that right for everyone.
```

```question
key: u04_l03_q02_speaking_right
type: multiple-choice
prompt: A class discussion gives students a chance to share ideas. Which responsibility fits this right?
choices:
  - Listen respectfully when others speak.
  - Talk over everyone else.
  - Hide the discussion question.
  - Say that only one person has rights.
correctAnswer: Listen respectfully when others speak.
explanation: The right to speak works with the responsibility to let others speak too.
hint: Choose the action that protects the discussion for everyone.
```

```question
key: u04_l03_q03_library_responsibility
type: passage-question
prompt: Read the scenario and answer.
passageTitle: Shared Library
passage: |
  Everyone in the town can borrow books from the public library.
  Books must be returned so other people can use them.
question: What responsibility goes with the right to borrow books?
choices:
  - Return the books so others can use them.
  - Keep every book forever.
  - Remove the library map.
  - Stop others from reading.
correctAnswer: Return the books so others can use them.
explanation: Returning books protects the shared right to use the library.
hint: Look for the action that helps other people use the same resource.
```
