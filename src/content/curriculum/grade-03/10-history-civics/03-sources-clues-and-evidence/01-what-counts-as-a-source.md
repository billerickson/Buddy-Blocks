## Teaching Goal

Expand the meaning of source beyond written texts.

## Student Outcome

The student can identify photographs, artifacts, documents, maps, timelines, and short texts as sources and name a clue each might provide.

## Misconception Checks

Thinking only books count as sources; believing artifacts cannot give information; treating all source details as equally useful for every question.

## Teaching Note

Place several "sources" side by side: a photo, an old tool, a map, a diary line, and a textbook sentence.

## Questions

```question
key: u03_l01_q01_source_recognition
type: multiple-choice
prompt: Which item could be used as a source about a town's past?
choices:
  - an old photograph of the town square
  - a story made up with no evidence
  - a blank piece of paper
  - a question with no clue
correctAnswer: an old photograph of the town square
explanation: A photograph can provide visual clues about people, buildings, or places.
hint: Choose the item someone can examine for clues.
```

```question
key: u03_l01_q02_source_clues
type: match-pairs
prompt: Match each source type to a clue it might provide.
pairs:
  - left: photograph
    right: what people or places looked like
  - left: artifact
    right: what tool or object people used
  - left: map
    right: where places were located
  - left: diary
    right: what one person wrote or felt
explanation: Different sources can answer different kinds of questions.
hint: Ask what kind of information each source can show.
```

```question
key: u03_l01_q03_museum_artifact
type: passage-question
prompt: Read the source description and answer.
passageTitle: Museum Object
passage: |
  A museum label says: This iron cooking pot was used by a family about 150 years ago.
question: What kind of source is the cooking pot?
choices:
  - artifact
  - document
  - photograph
  - secondary source
correctAnswer: artifact
explanation: An artifact is an object people made or used.
hint: The pot is an object from the past.
```

```question
key: u03_l01_q04_sources_as_evidence
type: fill-blank
prompt: Complete the sentence.
sentenceBefore: Historians use sources as
sentenceAfter: to answer questions about the past.
choices:
  - evidence
  - guesses
  - memories without clues
  - decorations
correctAnswer: evidence
explanation: Sources can provide evidence, or clues that support an answer.
hint: The word means information that supports a claim.
```

```question
key: u03_l01_q05_map_question
type: multiple-choice
prompt: Which question would a map most likely help answer?
choices:
  - Where did the road go?
  - What did one child write in a diary?
  - What did a cooking pot feel like to hold?
  - Which song did people sing?
correctAnswer: Where did the road go?
explanation: Maps are especially useful for place, route, and location questions.
hint: Choose the question about location or movement.
```

```question
key: u03_l01_q06_source_detail
type: multiple-choice
prompt: A photograph shows children standing beside a one-room schoolhouse. Which detail could the photograph provide?
choices:
  - what the schoolhouse looked like
  - what every child thought
  - what song they sang later
  - what the teacher wrote years later
correctAnswer: what the schoolhouse looked like
explanation: A photograph can show visible details, but not every hidden fact.
hint: Choose something a viewer could see in the photograph.
```
