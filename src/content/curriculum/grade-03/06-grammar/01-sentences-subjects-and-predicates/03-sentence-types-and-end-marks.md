## Teaching Goal

Connect sentence purpose to end punctuation.

## Student Outcome

The student can identify statements, questions, exclamations, and commands and choose a fitting end mark.

## Misconception Checks

Using an exclamation point for every important sentence; using a period for all commands; identifying questions only by word order, not meaning.

## Teaching Note

Read the same base sentence with different purposes and show how the end mark changes reader voice.

## Questions

```question
key: u01_l03_q01_question_mark
type: fill-blank
prompt: Choose the end mark.
sentenceBefore: Where did you put the map
sentenceAfter: ""
choices:
  - .
  - "?"
  - "!"
correctAnswer: "?"
explanation: The sentence asks a question, so it needs a question mark.
hint: "Look at what the sentence is doing: asking, telling, or showing strong feeling."
```

```question
key: u01_l03_q02_match_type_mark
type: match-pairs
prompt: Match each sentence type to its usual end mark.
pairs:
  - left: statement
    right: period
  - left: question
    right: question mark
  - left: exclamation
    right: exclamation point
  - left: command
    right: period or exclamation point
explanation: Sentence purpose helps the writer choose the end mark.
hint: A command can be calm or urgent, so its end mark can change.
```

```question
key: u01_l03_q03_find_command
type: multiple-choice
prompt: Which sentence is a command?
choices:
  - Did you close the gate?
  - The gate is closed.
  - Close the gate before dinner.
  - What a heavy gate!
correctAnswer: Close the gate before dinner.
explanation: This sentence tells someone to do something, so it is a command.
hint: A command gives a direction.
```

```question
key: u01_l03_q04_fix_end_mark
type: error-correction
prompt: Correct the end punctuation.
sentence: What a bright star?
acceptedAnswers:
  - What a bright star!
explanation: This sentence shows strong feeling, so an exclamation point fits.
hint: This is not asking for information.
```

```question
key: u01_l03_q05_note_punctuation
type: passage-question
prompt: Read the note and answer.
passageTitle: A Short Note
passage: |
  Please bring your sketchbook
question: Which end mark best completes the note?
choices:
  - .
  - "?"
  - "!"
correctAnswer: .
explanation: This is a calm command, so a period is the best end mark.
hint: Commands do not always need strong feeling.
```

```question
key: u01_l03_q06_write_sentence_type
type: multiple-choice
prompt: Which question about a book is correctly punctuated?
choices:
  - Where is the book?
  - Where is the book.
  - The book is on the shelf?
  - Read the book?
correctAnswer: Where is the book?
explanation: A question asks for information and ends with a question mark.
hint: Start with a question word such as who, what, where, when, why, or how.
```
