## Teaching Goal

Teach that some past-tense verbs use `-ed` while common irregular verbs change in other ways.

## Student Outcome

The student can choose regular and common irregular past-tense forms in short sentences.

## Misconception Checks

Adding `-ed` to every past verb; treating irregular forms as random with no need to learn; using two past markers such as `did went`.

## Teaching Note

Sort past verbs into `adds -ed` and `changes another way`.

## Questions

```question
key: u03_l03_q01_present_past_pairs
type: match-pairs
prompt: Match each present-tense verb to its past-tense form.
pairs:
  - left: walk
    right: walked
  - left: go
    right: went
  - left: run
    right: ran
  - left: see
    right: saw
explanation: Some past-tense verbs add -ed, and some common verbs change another way.
hint: Say each pair in a yesterday sentence.
```

```question
key: u03_l03_q02_choose_went
type: fill-blank
prompt: Choose the standard past-tense verb.
sentenceBefore: Last night, Dad
sentenceAfter: to the store.
choices:
  - go
  - goed
  - went
  - going
correctAnswer: went
explanation: The past tense of go is went, not goed.
hint: Go is an irregular verb.
```

```question
key: u03_l03_q03_fix_overregular
type: error-correction
prompt: Correct the verb form.
sentence: The child runned across the grass.
acceptedAnswers:
  - The child ran across the grass.
explanation: The past tense of run is ran.
hint: Run changes form in the past instead of adding -ed.
```

```question
key: u03_l03_q04_did_plus_base
type: multiple-choice
prompt: Which sentence uses did correctly?
choices:
  - I did saw the comet.
  - I did see the comet.
  - I did seen the comet.
  - I did seeing the comet.
correctAnswer: I did see the comet.
explanation: After did, use the base form see.
hint: Did already shows past time.
```
